Maven Wrapper Plugin
====================================================
Auto detect Maven wrapper for project.

### Attention

* No download support. please execute ./mvnw clean to download Maven first.


### Maven Wrapper usage

* Generate maven wrapper

```bash
mvn -N io.takari:maven:wrapper -Dmaven=3.5.2
```

* run maven wrapper

```bash
./mvnw clean
```
